# Bisht Associates - Law Firm Website

A modern, responsive website for Bisht Associates, a premier law firm specializing in appellate advocacy before the Supreme Court of India and High Courts.

## Features

- 🎨 Modern, professional design with Tailwind CSS
- 📱 Fully responsive layout (mobile, tablet, desktop)
- ⚡ Fast performance with Vite
- 🎯 Smooth scroll navigation
- 📧 Contact form with email integration
- 🏛️ Comprehensive practice areas showcase
- 💼 Professional About section

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icon library

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository or download the files
2. Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:3000`

### Build for Production

Create an optimized production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Sections

1. **Hero** - Eye-catching introduction with call-to-action
2. **About** - Firm overview and core values
3. **Courts** - Practice forums (Supreme Court & High Courts)
4. **Services** - Six practice areas with detailed descriptions
5. **Contact** - Service request form with contact information
6. **Footer** - Additional navigation and legal links

## Customization

### Update Contact Information

Edit the contact details in `src/App.jsx`:
- Email addresses (search for `@bishtassociates.com`)
- Phone numbers
- Office address

### Modify Services

Update the `services` array in `src/App.jsx` to add/edit practice areas.

### Change Colors

The site uses Tailwind's slate color palette. To change the theme, replace `slate` classes throughout the component.
